# Roadmap

AI D Teach is building toward a local-first, reusable AI tutoring workspace for parents, children, and educators.

## Near Term

- Improve child-friendly free conversation and tutor interruption behavior.
- Add more reusable interactive canvas activities beyond fractions and photosynthesis.
- Add automated UI smoke tests for lesson playback, voice states, and saved progress.
- Package setup so non-developers can run the local app and repository server more easily.
- Expand documentation for lesson schema, activity authoring, and local data privacy.

## Mid Term

- Add schema-validated AI lesson generation as an optional server-side feature.
- Add lesson versioning and review tools for the Ralph Loop.
- Add accessibility checks and keyboard-friendly canvas interactions.
- Add import/export for reusable lesson packs.
- Add more subject templates for math, reading, science, and vocabulary.

## Later

- Add optional cached higher-quality narration.
- Add classroom/teacher review workflows.
- Add durable storage options beyond JSON files.
- Add deployment guidance for trusted family or classroom environments.

## Not Planned By Default

- Always-on microphone recording.
- Hidden analytics or remote tracking.
- Client-side API key storage.
- Mandatory cloud AI dependency.
