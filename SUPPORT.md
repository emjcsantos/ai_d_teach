# Support

AI D Teach is an early open-source prototype.

Use GitHub issues for:

- Reproducible bugs.
- Lesson quality problems.
- Accessibility issues.
- Documentation gaps.
- Feature proposals.

Please include:

- Your operating system and browser.
- Steps to reproduce the issue.
- Expected behavior.
- Actual behavior.
- Whether the repository server was running.

Do not post API keys, `.env` files, child names, chat transcripts, or private learning data in public issues.
